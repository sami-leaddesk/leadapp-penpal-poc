# LeadApp Integration Library (prototype)

Javascript library to ease the integration with LeadDesk Application

## Usage

```javascript
import { LeadAppApi } from "leadapp-penpal-poc";

LeadAppApi().then((api) => {
  // Request full screen mode
  api.fullscreenEnable();
});
```
